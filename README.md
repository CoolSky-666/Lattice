## <a name="中文"></a>中文

### 项目简介
Lattice 是一个现代化的前端工具，专门为 **OpenMC** 中子输运代码设计。它的主要目的是为 OpenMC 提供毒物棒（Poison Pins）与控制棒（Control Rods）的列表布局生成。

该网页生成的坐标信息（如 `pos_x_96`, `pos_y_96` 与 `tube_x`, `tube_y`）可以直接用于 OpenMC 的栅格（Lattice）定义中，极大地简化了复杂组件的建模过程。

### OpenMC 使用示例
以下是如何在 OpenMC 代码中使用本项目生成的布局数据的示例：

```python
lat_A1 = openmc.RectLattice() 
lat_A1.lower_left = (-pitch*8.5, -pitch*8.5) 
lat_A1.pitch = (pitch, pitch) 
lat_A1.universes = np.tile(u_A1, (17, 17)) 
lat_A1.universes[tube_x, tube_y] = gt # 空导向管 

if pos_num == 0: 
    print("No Poision pins included!") 
elif pos_num == 64: 
    lat_A1.universes[pos_x_64, pos_y_64] = pt 
elif pos_num == 72: 
    lat_A1.universes[pos_x_72, pos_y_72] = pt 
elif pos_num == 80: 
    lat_A1.universes[pos_x_80, pos_y_80] = pt 
elif pos_num == 88: 
    lat_A1.universes[pos_x_88, pos_y_88] = pt 
elif pos_num == 96: 
    lat_A1.universes[pos_x_96, pos_y_96] = pt 
else: 
    raise ValueError("No such poision pin number is defined!!") 
```

## <a name="english"></a>English

### Project Overview
Lattice is a modern front-end tool designed specifically for the **OpenMC** neutron transport code. Its primary purpose is to provide layout lists for poison pins and control rods in OpenMC.

The coordinate information generated by this tool (such as `pos_x_96`, `pos_y_96` and `tube_x`, `tube_y`) can be directly applied to OpenMC's Lattice definitions, significantly simplifying the modeling process for complex assemblies.

